API List

List of APIs created using Python

1. Date difference

  • Calculates Date difference
  • Precision up to Seconds
  • Enter Start time and End time
  • Difference will be calcutated

2. Set Operations

  • Peforms Set Operations
  • Union Operation
  • Intersection Operation
  • Subtracction Operation

3. Matrix

  • Performs Matrix Operations
  • Transpose
  • Upper diagonal
  • Lower diagonal

4. Currency Words

  • Converts number currency to words
  • Enter data as number
  • Receive output as words

5. RSA encryption

  • Encrypts data using RSA
  • Public key, private key generated
  • Decryption using private key

6. Checksum

  • Generates checksum
  • MD5 checksum is used
  • Enter a alphanumeric string

7. Bar Code

  • Generated barcode for a given alphanumeric string
  • Code128 type is used to support alphanumeric

8. QR Code

  • Alphanumeric String must be entered
  • Image is generated for corresponding String

9. OTP

  • Random OTP will be generated
  • Otp size is variable
  • Phone and email support
  • Help center access

10. CAPTCHA

  • Generates Captcha
  • Provides the corresponding text
  • Provides the corresponding image
